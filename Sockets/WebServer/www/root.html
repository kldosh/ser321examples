You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /GCD -- returns the greatest common divisor of two numbers. For instance, /GCD?num1=45&num2=5 will return 5
  • /Isomorphic -- returns true if two input strings are isomorphic. Two words are considered isomorphic if the letters of the first word can be swapped out
    to create the second word. All occurances of a single letter must be swapped with the same letter, and no two letters can be swapped to the same character.
    For example: /Isomorphic?str1=egg&str2=add will return true, because the 'e' in "egg can be swapped with an 'a', and the 'g's can be swapped with 'd's
    to create "add"
